# 🦅 Eagle Pinball One

**Le flipper rétro-futuriste haute fidélité.**

Une expérience d'arcade immersive développée pour l'écosystème Farcaster, alliant esthétique Néon des années 80 et physique réaliste.

---

### 🕹️ Le Projet
`eagle-pinball-one` est le premier opus de la collection **Eagle Arcade**. L'objectif est de recréer les sensations authentiques des flippers mécaniques classiques directement dans une Frame ou un navigateur mobile.

### 🛠️ Spécifications Techniques
* **Moteur Physique :** Matter.js (Gravité, friction et restitution cinétique).
* **Design :** Full Neon Green / Cyberpunk.
* **Optimisation :** Conçu spécifiquement pour une expérience fluide sur mobile (format 400x600).

### 🚧 Statut du Développement : Alpha
- [x] Configuration de l'univers physique.
- [x] Gestion de la gravité et des collisions.
- [ ] Intégration des palettes (Flippers) mobiles.
- [ ] Système de bumpers réactifs et score.
